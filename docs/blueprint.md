# **App Name**: Project Pulse

## Core Features:

- Visual Progress Dashboard: Dashboard visualization to track project phases and task status, displaying real-time progress.
- Data Export: Allow CSV/Excel export of all task data.
- Report Generation: Automatic generation of a summary report by phase, assignee, and completion rate.
- Proactive Suggestion Engine: Provide proactive update suggestions based on detected progress stalls, using an AI tool that assesses task dependencies and suggests reassignments or deadline extensions.
- Task Management: Allow direct task reassignment with automatic updates to dependencies.
- Template Library: Offer customizable templates for SRS, testing reports, and deployment checklists.
- Iteration Management: Enable loop-back functionality for iterations, allowing transitions (e.g., from Monitoring back to Requirements).

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey reliability and professionalism.
- Background color: Light Gray (#ECEFF1), subtly desaturated indigo, for a clean interface.
- Accent color: Amber (#FFC107), analogous to indigo, for important notifications and CTAs.
- Body and headline font: 'Inter', a sans-serif with a modern look suitable for both headlines and body text.
- Use consistent, clear icons from a library like Material Design Icons, with slight animations for status updates.
- Kanban-style dashboard layout for easy task status updates. Use drag-and-drop functionality.
- Use subtle animations when updating task statuses, adding new tasks, or exporting data.
